# 📈 Apple Stock Time Series Analysis

## 📌 Steps Performed
- Converted `Date` to datetime index
- Visualized closing price trend
- Created 30-day and 100-day moving averages
- Performed trend decomposition
- Split data into train and test sets
- Built ARIMA forecasting model
- Evaluated forecast using RMSE

## 📊 Key Insights
- Apple stock shows a strong long-term upward trend
- 30-day moving average captures short-term price movement
- 100-day moving average highlights long-term trend
- ARIMA provides near-future stock price forecasting

## 🛠 Tools Used
- Python 
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- Scikit-learn
- Jupyter Notebook
